# Digital Personal Information Manager

A simple **console-based C++ program** for managing personal information. It allows you to add, view, and search records containing names, phone numbers, and email addresses. All data is stored persistently in a text file (`records.txt`).

## Features

- Add new personal records (Name, Phone, Email)  
- View all stored records  
- Search for a record by name  
- Save and load records automatically to/from a text file  

## Requirements

- C++ compiler (g++, MinGW or any other modern C++ compiler)  
- Basic command-line knowledge  
- Optional: Windows, Linux, or macOS  

## Follow the menu:

1. Add New Record
2. View All Records
3. Search Record
4. Exit
